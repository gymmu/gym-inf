## Webseiten gestalten

Das standart Layout das für Webseiten verwendet wird, ist oftmals nicht
angemessen und nicht schön. Zum Glück kann man die Darstellung mit `CSS` super
einfach anpassen. Wie man das macht, schauen wir uns in diesem Kapitel an.

<CSSIntroCode htmlCode='<h1 class="title">Hello World!</h1>' css={`.title {
    background-color: white;
    color: black;
    font-size: 32pt;
}`} />
