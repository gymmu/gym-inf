## Mehr zu Pfaden

Pfade sind das wichtigste und aber auch komplizierteste Element in `SVG`,
deshalb schauen wir uns die Pfade in diesem Kapitel nochmals genauer an.

<LearningGoals>

- Sie können die generelle Syntax von einem Pfad lesen.
- Sie wissen wie man einen Pfad automatisch schliesst.
- Sie können die Strichfarbe von einem Pfad anpassen.
- Sie können die Strichbreite von einem Pfad anpassen.
- Sie können erkennen aus welchen Teilen ein Pfad zusammengesetzt ist.
- Sie können selber weitere Teile an einen Pfad anhängen.
- Sie kennen die verschiedenen Zeichenbefehle für einen Pfad.
- Sie kennen den unterschied zwischen gross und klein geschriebenen
  Zeichenbefehlen.

</LearningGoals>
