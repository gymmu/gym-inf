## Seiteninhalt mit Javascript verändern

Eine der häufigsten Aktionen die man in Javascript macht, ist das verändern der
Webseite. Sollten sich Daten verändert haben, dann möchte man das auf der
Webseite neu so anzeigen, ohne diese neu laden zu müssen. Mit Javascript können
wir dafür das einfache Muster verwenden, das Sie im letzte Kapitel bereits
gesehen haben.

### Werte laden, verändern und zurückschreiben

Es gibt verschiedene Werte die man verändern kann, jeder Wert kommt jedoch von
einem HTML-Element. Damit wir diesen auch verwenden können, speichern wir das
HTML-Element zuerst in einer Variable, diese können wir dann im Code weiter
verwenden.

<Example title="Beispiel: Text anpassen">
Wir betrachten die folgende Funktion in Javascript, sie soll den Text von
einem Element mit dem Text `"Dieser Text wurde gelöscht!"`
ersetzen.

```javascript
function deleteText() {
  const elemToDeleteText = document.querySelector("#to-delete")
  elemToDeleteText.textContent = "Dieser Text wurde gelöscht!"
}
```

Die Funktion existiert nun, und wir müssen sie nur noch ausführen. Wir brauchen
aber auch noch das Element auf dem der Text gelöscht werden soll, das macht der
folgende HTML-Code.

```html
<p id="to-delete">Hier steht Text der später gelöscht werden soll.</p>

<button onclick="deleteText()">Text löschen</button>
```
