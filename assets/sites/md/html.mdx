## HTML

`HTML` ist eine Beschreibungssprache. Sie basiert auf `XML` und ist darum sehr ähnlich wie `SVG`. Die Sprache selber ist super einfach, alles ist in sogenannte Tags aufgeteilt. Ein Tag hat immer die folgende Syntax: `<tag-name>...</tag-name>`

## Text

Text sollte niemals direkt in eine Webseite geschrieben werden. Wir möchten den Text immer in ein Paragraph (`<p>`-Element) packen.

<PrismBlock>
    ```html
    <p>Hier ist irgend welcher Text.</p>
    ```
</PrismBlock>

Wenn wir mehrere Abschnitte haben, dann packen wir jeden Abschnitt in ein solches `<p>`-Element.

<PrismBlock>
    ```html
    <p>Dies ist der erste Abschnitt.</p>
    <p>Die ist der zweite Abschnitt.</p>
    ```
</PrismBlock>
</Slide>

## Abschnitte

Das moderne Web (HTML5 seit ca. 2010) hat verschiedene Elemente, die das logische benennen von Webseiten teilen einfacher machen sollen.

<PrismBlock>
    ```html
    <header><h1>Webseiten Titel</h1></header>

    <main>
        <p>Das ist der Hauptteil</p>
        <section>
            <p>Das ist ein Abschnitt im Hauptteil</p>
        </section>
        <article>
            <p>Das ist ein Artikel im Hauptteil</p>
        </article>
    </main>

    <footer>Hier ist eine Fusszeile</footer>
    ```
</PrismBlock>

<Slide>
    ## Kommentare

    Kommentare sind Stellen im Code die nicht auf der Webseite angezeigt werden. Sie helfen uns zu verstehen was wir programmieren wollten, oder wenn wir noch etwas ändern werden.

    <PrismBlock>
      ```html
      <aside>
        <nav>
        <!-- Hier wird die Navigationsliste erstellt,
             sobald wir wissen wie man Listen macht.
         -->
        </nav>
      <aside>
      ```
    </PrismBlock>
</Slide>

<Slide>
    ## Listen

    In `HTML` gibt es 2 Arten von Listen, die nummerierten und die nicht-nummerierten. Beide Listen haben aber jeweils die gleichen Listen-Einträge.

    <PrismBlock>
      ```html
        <ul>
          <li>Home</li>
          <li>About</li>
          <li>Hobbies</li>
          <li>Kontakt</li>
        </ul>
      ```
    </PrismBlock>

    oder als nummerierte Liste

    <PrismBlock>
    ```html
      <ol>
        <li>Erster Eintrag</li>
        <li>Zweiter Eintrag</li>
        <li>...</li>
      </ol>
    ```
    </PrismBlock>
</Slide>

<Slide>
    ## Links

    Auf Webseiten, möchte man oftmals auf andere Webseiten oder auf Teile der eigenen Webseite verweisen. Dafür verwendet man das Anker-Element.

    <PrismBlock>
      ```html
      <!-- Externer Link -->
      <a href="https://www.gym-muttenz.ch">Gym Muttenz</a>

      <!-- Interner Link -->
      <a href="./unterseiten/hobbies.html">Hobbies</a>

      <!-- Link zu einem Seitenelement -->
      <a href="#home">Home</a>
      ```
    </PrismBlock>
</Slide>


<Slide>
    ## Navigationsleisten

    Navigationsleisten auf Webseiten müssen immer selbst erstellt werden. Dafür verwendet man Listen mit Links darin. Damit es für Screenreader und Suchmaschinen einfacher ist, verwendet man auch semantische Elemente.

    <PrismBlock>
      ```html
      <aside>
        <nav>
          <ul>
            <li>
              <a href="#home">Home</a>
            </li>
            <li>
              <a href="#about">About</a>
            </li>
            <li>
              <a href="#hobbies">Hobbies</a>
            </li>
            <li>
              <a href="#contact">Kontakt</a>
            </li>
          </ul>
        </nav>
      </aside>
      ```
    </PrismBlock>
</Slide>

<Slide>
    ## Attribute

    Alle `HTML`-Elemente können Attribute haben. Einige Elemente brauchen diese Attribute auch, um zu funktionieren.

    <PrismBlock>
      ```html
        <main>
          <section id="home">
            <h2>Home</h2>
            <p>Dies ist die Home-Page</p>
          </section>
          <section>
            <h2>Seitenlinks</h2>
            <p>
              Dieser Link bringt und zurück zu <a href="#home">Home</a>.
            </p>
          </section>
          <section>
            <h2>Externe Links</h2>
            <p>
              Dieser Link öffnet
              <a href="https://www.gym-muttenz.ch" target="_blank">Gym Muttenz</a>
              in einem neuen Tab.
            </p>
          </section>
        </main>
      ```
    </PrismBlock>
</Slide>

    ## Inline und Block Elemente

    `HTML`-Elemente lassen sich generell in 2 Arten von Elementen unterteilen. Wir haben die Inline-Elemente und die Block-Elemente. Der unterschied ist sehr einfach, eine Inline-Element, kann direkt in den Text integriert werden. Ein Block-Element nutzt immer die ganze Breite.

    ### Generelles Block-Element

    <PrismBlock>
      ```html
        <div class="frame-container">
          Text mit einem Rahmen.
        </div>
      ```
    </PrismBlock>

    ### Generelles Inline-Element

    <PrismBlock>
      ```html
        <span class="mark">
          Dieser Text soll hervorgehoben werden.
        </span>
      ```
    </PrismBlock>
Weiter unten finden Sie noch mehr Inline- und Block-Elemente.
</Slide>

## Elemente für Formulare

- `<input type="Typen">` mit verschiedenen Typen:
    - `text` - Einzeiliges Texteingabefeld
    - `password` - Eingabefeld für Passwörter
    - `email` - Eingabefeld für E-Mail-Adressen
    - `number` - Eingabefeld für Zahlen
    - `date` - Eingabefeld für Datumsangaben
    - `checkbox` - Kontrollkästchen
    - `radio` - Optionsfeld
    - `file` - Datei-Upload
    - `submit` - Senden-Button
    - `reset` - Zurücksetzen-Button
    - `range` - Slider in einem Bereich
- `<textarea>` - Mehrzeiliges Texteingabefeld
- `<select>` - Dropdown-Liste
  - `<option>` - Optionen innerhalb der Dropdown-Liste
- `<button>` - Allgemeiner Button
- `<label>` - Beschriftung für Formularelemente
- `<fieldset>` - Gruppiert verwandte Elemente
- `<legend>` - Beschriftung für ein `fieldset`
- `<datalist>` - Vorgaben für `<input>`-Felder
- `<output>` - Ausgabe von Berechnungen oder Ergebnissen
