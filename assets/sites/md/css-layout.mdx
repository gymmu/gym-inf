## Einfache Layouts

Oftmals möchten Sie gerne mehrere Elemente in einer Zeile haben, welche dann
alle gleich breit sind, und die Abstände dazwischen automatisch angepasst
werden. Das kann einfach mit dem **Flexbox** Modell von CSS erreicht werden.

<Example title="Beispiel: Navbar">

Auf vielen Webseiten wird eine Navbar am oberen Rand verwendet. In diesem
Beispiel versuchen wir so eine Navbar nach zubauen, und zu verstehen wie das
funktioniert.

Erstellen Sie zuerst eine Liste mit 3 Links darin.

```html
<ul class="navbar">
  <li><a href="/">Home</a></li>
  <li><a href="#section-1">Abschnitt 1</a></li>
  <li><a href="#section-2">Abschnitt 2</a></li>
</ul>
```

Diese Liste sieht noch schrecklich aus, und überhaupt nicht so wie wir das
brauchen. Mit dem folgenden CSS Code kann das aber einfach geändert werden.

Zuerst sagen wir dass die Liste horizontal angezeigt werden soll, und dass alle
Elemente gleichmässig verteilt werden.

```css
.navbar {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: space-between;
  list-style: none;
  margin: 0;
  padding: 20px 0;
}
```

Das `display: flex;` mit dem restlichen Code, sorgt dafür dass die Elemente
entsprechend automatisch angeordnet werden. Wir müssen keine Breite dafür
angeben, müssen aber aufpassen wenn die Liste zu gross wird, oder die Elemente
zu breit werden. Das kann auch schlecht aussehen wenn es auf dem Smartphone
angeschaut wird.

Wir müssen dann noch die Stiele für die Links angeben. Das passen Sie am besten
in dem folgenden Selektor an: `.navbar > li > a`
