Internet Quiz

Testen Sie Ihr Wissen über das Internet, Datenübertragung und Sicherheit!

Dieses Quiz basiert auf den Inhalten des Internet-Kapitels. Falls Sie Schwierigkeiten mit den Fragen haben, schauen Sie sich das Kapitel noch einmal an.

Was ist die Hauptaufgabe eines Routers?

  1. Daten verschlüsseln
  2. Verschiedene Netzwerke miteinander verbinden
  3. Webseiten speichern
  4. Viren scannen

Alle Daten im Internet werden standardmässig verschlüsselt übertragen.

  1. Wahr
  2. Falsch

Was bedeutet das 'S' in HTTPS?

  1. Server
  2. Secure
  3. Speed
  4. System

In welche Teile werden Daten bei der Übertragung im Internet aufgeteilt?

  1. Bytes
  2. Pakete
  3. Chunks
  4. Frames

Ihr ISP (Internet Service Provider) kann sehen, welche Webseiten Sie besuchen.

  1. Wahr
  2. Falsch

Wann sollten Sie besonders auf HTTPS achten?

  1. Nur beim Online-Banking
  2. Beim Einloggen und bei sensiblen Daten
  3. Nur auf unsicheren Webseiten
  4. Nie, HTTP ist auch sicher

In öffentlichen WLANs können andere Personen Ihren Datenverkehr abfangen.

  1. Wahr
  2. Falsch

Was ist NAT (Network Address Translation)?

  1. Ein Verschlüsselungsprotokoll
  2. Eine Methode um private IP-Adressen in öffentliche zu übersetzen
  3. Ein Browser-Plugin
  4. Ein Virenscanner

Mit HTTPS kann der ISP sehen, was Sie auf einer Webseite genau tun.

  1. Wahr
  2. Falsch

Wie erkennen Sie eine sichere HTTPS-Verbindung im Browser?

  1. An der grünen Adressleiste
  2. Am Vorhängeschloss-Symbol in der Adresszeile
  3. An der Geschwindigkeit der Webseite
  4. Am Logo der Webseite

Zusammenfassung der Lernziele

Nach Abschluss dieses Quiz sollten Sie:

  • ✅ Verstehen wie Daten im Internet übertragen werden
  • ✅ Wissen dass unverschlüsselte Übertragungen öffentlich einsehbar sind
  • ✅ Den Unterschied zwischen HTTP und HTTPS kennen
  • ✅ Wissen wann Daten sicher versendet werden
  • ✅ Die Rolle von Router und ISP verstehen
  • ✅ Sicherheitsrisiken in öffentlichen WLANs kennen

Haben Sie Fragen oder möchten Sie mehr über ein bestimmtes Thema erfahren? Schauen Sie sich das Internet-Kapitel noch einmal an!
