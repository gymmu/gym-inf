Animationen in SVG

Animationen werden im Web recht häufig verwendet, damit es für Besucher einfacher ist zu erkennen, wo gerade etwas auf der Webseite passiert. Gute Animationen sind oftmals sehr komplex, aber bereits einfache Animationen können schon sehr viel ausmachen. Zum Beispiel ein Hamburger-Menu, das anzeigt ob die Seitenleiste geöffnet oder geschlossen ist. Wie man eine solche Animation machen könnte, schauen wir uns hier an.

Lernziele

  • Sie wissen wie man eine einfache Animation in SVG erstellt.
  • Sie wissen welche Attribute man animieren kann.
  • Sie kennen 2 Arten der Animation (Attribute und Transformationen).
  • Sie wissen wie das values-Attribut bei Animationen funktioniert.

Attribute animieren

Animationen müssen nicht schwierig sein. Man kann bereits viele Effekte erreichen, wenn schon nur ein einziges Attribut animiert wird. Man kann zum Beispiel sagen, das sich ein Kreis hin und her bewegen soll.

Wählen Sie ein Attribut aus der Liste aus. Ändern Sie die Animationswerte, so das die Animation sinn ergibt, und fügen Sie neue Werte hinzu. Schauen Sie sich genau an wie der SVG-Code aufgebaut ist.

Weitere Objekte animieren

Wir können natürlich auch andere Objekte animieren, nicht nur Kreise. Mit dem animate-Element, sagen Sie welches Attribut von dem direkten Elternelement animiert werden soll. Das geht für alle möglichen Elemente. Sie können sogar das d-Attribut vpn Pfaden animieren. Diese Aufgabe lassen wir aber für die Übungen. Nun möchten wir uns zuerst mit der Animation von Transformationen beschäftigen, denn hier können wir auch einfach ganze Gruppen animieren.

Transformationen animieren

Wenn wir Transformationen animieren, dann können wir alle Elemente in eine Gruppe (ein g-Element packen). Dann gilt die Animation für die ganze Gruppe. Ein Animations-Element verändert immer das direkte Elternelement. Da Transformationen auf alle Elemente angewendet werden können, ist es sehr einfach ganze Gruppen zu animieren.

Wählen Sie ein Animationsattribut aus. Überlegen Sie sich immer zuerst wie die Animation aussehen könnte. Ändern Sie dann die Animationswerte so, dass die Animation sinn ergibt. Verschieben Sie auch den Animationsursprung.
