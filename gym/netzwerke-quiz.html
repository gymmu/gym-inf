Quiz: Netzwerkkomponenten und -sicherheit

Testen Sie Ihr Wissen über Netzwerkgeräte, Subnetting, NAT, Firewalls und Port-Forwarding!

Dieses Quiz basiert auf den Inhalten des Netzwerke-Kapitels. Falls Sie Schwierigkeiten mit den Fragen haben, schauen Sie sich das Kapitel noch einmal an.

Was ist der wesentliche Unterschied zwischen einem Hub und einem Switch?

  1. Ein Switch leitet Pakete gezielt an den richtigen Port weiter (anhand der MAC-Adresse), ein Hub sendet an alle.
  2. Ein Hub ist schneller als ein Switch.
  3. Ein Switch arbeitet mit IP-Adressen, ein Hub mit MAC-Adressen.
  4. Ein Hub kann mehrere Netzwerke verbinden, ein Switch nicht.

Wie viele nutzbare Hosts gibt es in einem /28-Subnetz?

  1. 14 Hosts
  2. 16 Hosts
  3. 28 Hosts
  4. 30 Hosts

Was speichert ein Router in seiner Routing-Tabelle?

  1. Welche Netzwerke über welchen nächsten Hop (Next Hop) erreichbar sind.
  2. Eine Liste aller verbundenen Geräte mit ihren MAC-Adressen.
  3. Alle besuchten Webseiten der Netzwerkteilnehmer.
  4. Die IP-Adressen aller DNS-Server.

Wie funktioniert NAT beim Weiterleiten von Antwortpaketen an das richtige Gerät im lokalen Netz?

  1. Durch eine NAT-Tabelle, die interne IP:Port auf externe IP:Port abbildet, und beim Eingang die Übersetzung umkehrt.
  2. Das Zielgerät sendet regelmässig eine Anfrage, und der Router merkt sich die letzte Anfrage.
  3. Alle Antwortpakete werden an alle Geräte im Netzwerk gesendet.
  4. NAT verwendet DNS, um das richtige Gerät zu finden.

NAT allein ist ein ausreichender Ersatz für eine Firewall.

  1. Wahr
  2. Falsch – NAT bietet indirekten Schutz, ist aber kein vollwertiger Ersatz für eine Firewall.

Was ist der Vorteil einer 'stateful' Firewall gegenüber einer 'stateless' Firewall?

  1. Sie verfolgt den Zustand von Verbindungen und erlaubt Antwortpakete automatisch, ohne explizite Regeln.
  2. Sie ist schneller, weil sie keinen Zustand speichern muss.
  3. Sie blockiert alle eingehenden Verbindungen.
  4. Sie kann nur HTTP-Verkehr filtern.

Wofür wird Port-Forwarding benötigt?

  1. Um von aussen auf einen Server im lokalen Netzwerk zugreifen zu können, der hinter NAT versteckt ist.
  2. Um die Internetgeschwindigkeit zu erhöhen.
  3. Um DNS-Anfragen zu beschleunigen.
  4. Um WLAN-Geräte mit dem kabelgebundenen Netz zu verbinden.

Was ist eine DMZ (Demilitarized Zone) in der Netzwerkarchitektur?

  1. Ein Netzwerksegment zwischen Internet und internem Netz für öffentlich zugängliche Server, das bei Kompromittierung das interne Netz schützt.
  2. Ein Netzwerk ohne Sicherheitsregeln für Tests.
  3. Eine Zone, in der keine Verschlüsselung verwendet wird.
  4. Ein VPN-Server für externe Mitarbeitende.

WPA2 mit einem starken Passwort gilt heute noch als sicher für WLAN-Verschlüsselung.

  1. Wahr – WPA2 (AES) ist weiterhin sicher bei einem starken Passwort.
  2. Falsch – WPA2 ist vollständig veraltet und unsicher.

Was ist ein VLAN und wozu dient es?

  1. Ein virtuelles LAN, das ein physisches Netzwerk logisch in getrennte Segmente aufteilt, ohne separate Hardware.
  2. Ein VPN für lokale Netzwerke.
  3. Ein drahtloses Netzwerk mit erhöhter Reichweite.
  4. Ein Netzwerkprotokoll für virtuelle Maschinen.

Welche IP-Adresse gehört zum privaten Adressbereich gemäss RFC 1918?

  1. 172.20.0.1
  2. 8.8.8.8
  3. 142.250.74.14
  4. 1.1.1.1

Was ist ein DNS-Sinkhole?

  1. Ein lokaler DNS-Server, der bekannte Werbe- und Tracking-Domains mit einer ungültigen IP beantwortet und so Werbung netzwerkweit blockiert.
  2. Ein Angriff, der DNS-Anfragen auf gefälschte Server umleitet.
  3. Ein DNS-Server, der alle Anfragen verwirft.
  4. Eine Methode, DNS-Anfragen zu verschlüsseln.

  • Den Unterschied zwischen Hub, Switch und Router erklären können
  • Einfache Subnetting-Berechnungen durchführen können
  • Die NAT-Tabelle und ihre Funktion beschreiben können
  • Stateless vs. stateful Firewalls unterscheiden können
  • Port-Forwarding konfigurieren und seine Sicherheitsimplikationen kennen
